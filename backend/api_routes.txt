# API Routes Documentation

## Auth Routes
POST /auth/register 
POST /auth/login 
POST /auth/logout 
POST /auth/refresh
POST /auth/forgot
POST /auth/reset
GET /auth/profile 
PUT /auth/profile 

## Curd Routes
POST /curd/create - Create a new todo  
GET /curd/getAll - Get all todos for logged in user  
PUT /curd/update/:id - Update a specific todo  
DELETE /curd/delete/:id - Delete a specific todo  

## Test Route
GET /test - Test route to check server status
